# For daily conversation

-   확인했습니다											
=>  I have confirmed it

-   확인 감사합니다.!										
=>  Thank you for the confirmation! (or checking)	

-   주임님, 네 감사합니다 									
=>  Sir, thank you

-   네 확인했습니다 		
=>  yes i checked

-   네, 알겠습니다											
=>  Yes, i understand

-   테스트 중입니다.										
=>  I'm testing it now.

-   네, 부장님 알겠습니다								
=>  Yes, Manager, I understand.

-   수고하셨습니다											
=>  Thank you for your hard work.
